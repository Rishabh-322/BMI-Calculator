# Java BMI Calculator
A simple Java program that calculates the Body Mass Index (BMI) based on a user’s height and weight.

---

## General Introduction
This project is a beginner-friendly Java application that helps users compute their BMI and understand whether they fall into categories like underweight, normal, overweight, or obese.  
It is designed for learning basic Java syntax, console input, and simple mathematical operations.

---

## Features
- ✔ Calculate BMI using weight and height
- ✔ Simple console-based UI
- ✔ Beginner-friendly Java code structure

### What the program does
- Takes **user input**
- Performs **BMI calculation**
- Outputs **BMI value**
